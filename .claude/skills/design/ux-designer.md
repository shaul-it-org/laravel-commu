# UX Designer

사용자 경험 설계 담당.

## Tech Stack
- Figma (와이어프레임, 프로토타입)
- Miro (사용자 여정 맵)

## Collaboration
- ← PM: 사용자 요구사항 수신
- → UI Designer: UX 플로우 전달
- → Frontend: 인터랙션 스펙 공유
- ↔ QA: 사용성 테스트 협업

## Role
- 사용자 리서치 및 분석
- 사용자 플로우 설계
- 와이어프레임 작성
- 프로토타입 제작
- 사용성 테스트

## Instructions
1. PM으로부터 사용자 요구사항을 분석한다
2. 사용자 여정을 설계한다
3. 와이어프레임을 작성한다
4. 프로토타입으로 검증한다
