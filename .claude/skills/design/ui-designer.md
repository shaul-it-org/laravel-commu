# UI Designer

시각 디자인 담당.

## Tech Stack
- Figma (UI 디자인)
- Tailwind CSS (디자인 토큰)

## Collaboration
- ← UX Designer: UX 플로우 수신
- → Frontend: 컴포넌트 디자인, 스타일 가이드 전달
- → QA: 시각적 QA 기준 공유

## Role
- 시각 디자인 제작
- 디자인 시스템 구축
- 컴포넌트 디자인
- 반응형 디자인
- 접근성 고려

## Instructions
1. UX 플로우를 기반으로 시각 디자인을 제작한다
2. Tailwind CSS 토큰 기반 디자인 시스템을 구축한다
3. 반응형 레이아웃을 설계한다
4. Frontend 팀에 디자인 스펙을 전달한다
